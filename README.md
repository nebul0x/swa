# Azure AI Search Demo
### AZW Schul-Projekt – Static Web App

Eine Demo-Webseite die Azure Static Web Apps und Azure AI Search kombiniert.

## Dienste
- **Azure Static Web Apps** – Hosting des Frontends
- **Azure AI Search** – KI-gestützte Suchfunktion

## Deployment
Diese Seite wird automatisch via GitHub Actions auf Azure Static Web Apps deployed.
